# ResumeAI 🚀

AI-powered Resume Analyzer & Job Recommendation Platform built using MERN Stack.

## Features

- Resume Upload (PDF)
- AI Resume Analysis
- ATS Score Checker
- Resume Improvement Tips
- Missing Skills Detection
- AI Job Recommendation
- JWT Authentication
- MongoDB Database
- RapidAPI Job Search Integration
- Groq AI Integration

---

## Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM
- React Toastify

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer

### AI & APIs
- Groq AI
- RapidAPI JSearch API

## Installation

### Clone Repository

```bash
git clone https://github.com/singhalok999/resumeai.git
```

---

### Install Dependencies

```bash
npm install
cd client
npm install
cd ../server
npm install
```

---

## Environment Variables

Create `.env` inside server folder:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
GROQ_API_KEY=your_groq_api_key
RAPID_API_KEY=your_rapid_api_key
```

---

## Run Project

### Backend

```bash
cd server
npm run dev
```

### Frontend

```bash
cd client
npm start
```
